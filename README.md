- **White/Light Mode:** Optimizes for a professional, CAD-software feel.

## Fixing Deployment Errors (404)

If you see a `Failed to create deployment (status: 404)` error in GitHub Actions, follow these steps to enable the hosting:

1. Go to your repository **Settings** on GitHub.
2. Select **Pages** from the left sidebar.
3. Under **Build and deployment** > **Source**, change the dropdown to **GitHub Actions**.
4. Once selected, GitHub will automatically use the `.github/workflows/deploy-pages.yml` file to host your site.
